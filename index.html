SIMPLE CSS


¿qué es?

Es una librería (CSS3) para el pre-formateo de estilos en paginas web, que trabaja bajo el patrón de arquitectura de software CtrlERA , creando un estilo simple en las etiquetas básicas de HTML / Logo


instalación

<link rel="stylesheet" type="text/css" href="base/E/css/simple/simple.css"/>
<script type="text/javascript" src="base/E/js/simple/simple.js"></script>
